# 🌧️ Rainz.net

The free AI-powered weather PWA that helps you stay dry.

## What is Rainz?

Rainz is a free AI-powered Progressive Web App that gives you hyper-accurate weather forecasts by aggregating 12 weather sources alongside user-submitted hyperlocal data. No ads, no paywalls.

Download it at rainz.net/download

## Features

**DryRoutes** — Plan rain-free walking and cycling routes with intelligent departure timing. DryRoutes analyses rain patterns along your exact path so you stay dry from door to door.

- Go Mode — one-tap navigation to the driest route
- AR Navigation — augmented reality route overlay
- Voice Navigation — hands-free guidance
- Rain Radar — live precipitation along your route
- Departure Picker — find the driest window to leave
- Carbon & Calorie Tracker
- Umbrella Score — instant yes/no on whether to grab an umbrella
