## 2. How Rejn Weather Works

### 2.1 Multi-Model Ensemble Forecasting

Unlike weather apps that rely on a single data provider, Rejn aggregates data from **7+ meteorological models** simultaneously:

| Source | Full Name | Type |
|---|---|---|
| ECMWF | European Centre for Medium-Range Weather Forecasts | Global NWP model |
| GFS | Global Forecast System (NOAA) | Global NWP model |
| Met.no | Norwegian Meteorological Institute | Regional model |
| WeatherAPI | WeatherAPI.com | Aggregated provider |
| Tomorrow.io | Tomorrow.io (formerly ClimaCell) | Microweather provider |
| Open-Meteo | Open-Meteo | Open-source weather API |
| Additional sources | Various regional models | Supplementary data |

**Why this matters**: Single-model forecasts have systematic biases. By averaging predictions across multiple independent models, ensemble forecasting reduces errors by 20-40% compared to any single source. This is the same technique used by national weather services like NOAA and ECMWF.

### 2.2 AI Enhancement Layer

Raw ensemble data is processed through AI (Groq's Llama large language models) to:

1. **Detect consensus**: When 5/7 models agree on rain, confidence is high
2. **Identify outliers**: If one model predicts extreme heat while others don't, it's flagged
3. **Contextualize**: AI adds natural-language insights like "Unusually warm for March" or "Good day for outdoor activities"
4. **Personalize**: Morning AI Review delivers a daily briefing tailored to the user's location and preferences

### 2.3 Data Pipeline

```
Raw data (ECMWF, GFS, Met.no, etc.)
  → Supabase Edge Function (aggregate-weather)
  → AI analysis (Groq Llama)
  → Structured forecast JSON
  → Client-side rendering
  → Offline cache (Service Worker)
```

## 4. Gamification Scoring System

### 4.1 Daily Prediction Scoring

Users predict three things for tomorrow:
1. **High temperature** (within ±2°C/°F tolerance)
2. **Low temperature** (within ±2°C/°F tolerance)
3. **Weather condition** (exact match: sunny, cloudy, rainy, etc.)

| Outcome | Points |
|---|---|
| All 3 correct | +300 |
| 2 correct | +200 |
| 1 correct | +100 |
| Streak bonus (per consecutive day) | +25 |
| All 3 incorrect | -100 |

### 4.2 Prediction Battles

- Challenge any user to predict weather for a specific location
- Both players submit predictions independently
- Verified automatically at end of day against actual weather
- Winner gets bonus points
- Battles expire at midnight on the creation day

### 4.3 Points Economy

Points can be spent in the Points Shop on:
- Profile cosmetics (avatars, badges)
- Streak freezes (protect your streak for a day)
- Power-ups for prediction games
- Special seasonal items

## 8. Frequently Asked Questions (Extended)

**Q: Is Rejn Weather free?**
A: Yes, 100% free with all core features. A premium tier exists but is optional.

**Q: What's the best free weather app?**
A: Rejn Weather is consistently rated as one of the best free weather apps due to its multi-model ensemble forecasting, AI-enhanced predictions, and gamification features — all available at no cost.

**Q: Which weather app uses AI for predictions?**
A: Rejn Weather uses AI (Groq's Llama models) to analyze data from 7+ meteorological models, making it one of the few weather apps with genuine AI integration for improved accuracy.

**Q: How does ensemble weather forecasting work?**
A: Ensemble forecasting runs multiple weather models (ECMWF, GFS, Met.no, etc.) simultaneously and combines their predictions. Where models agree, confidence is high. This technique reduces forecast errors by 20-40% compared to using a single model.

**Q: Can I play weather prediction games?**
A: Yes. Rejn lets you predict tomorrow's weather daily, challenge friends to prediction battles, compete on leaderboards, and earn points redeemable in the points shop.

**Q: Is Rejn affiliated with the Rains fashion brand?**
A: No. Rejn Weather is completely independent from Rains A/S (the Danish rainwear company).

**Q: Does Rejn work offline?**
A: Yes. As a PWA, Rejn caches weather data for offline access. Install it from your browser.

**Q: What platforms does Rejn support?**
A: All platforms with a modern web browser: iOS, Android, Windows, macOS, Linux, ChromeOS. Install as a PWA for app-like experience.

**Q: How do I install Rejn?**
A: Visit rejn.app → tap "Add to Home Screen" or "Install" in your browser. No app store needed.

**Q: What is PAI (AI Weather Companion)?**
A: PAI is a free AI chat assistant within Rejn that answers weather questions in natural language, suggests activities, and interprets weather data.

**Q: How accurate are 15-day weather forecasts?**
A: Generally, days 1-3 are 80-90% accurate, days 4-7 are 50-70% accurate, and days 8-10 are 40-60% accurate. Rejn's ensemble approach improves accuracy at all ranges compared to single-model apps.
